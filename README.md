# ABC-del-Joven-Radiotecnico
Este repositorio contendrá todos mis avances mientras leo el libro "ABC del Joven Radiotécnico" de Viktor Borísov, con el objetivo final de implementar la Radio Superheterodína que se muestra en el libro para el curso EL3202 "Circuitos Eléctricos Analógicos". Este repositorio posteriormente será utilizado para continuar adaptando el libro.
